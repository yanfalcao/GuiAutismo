## **Descrição**

Aplicativo feito para meu TCC em 2021 para o curso de Análise e Desenvolvimento de Sistemas. O aplicativo é pensado para o público autista, onde o aplicativo permite a avaliação, apresentação de dados e notas de acessibilidade em cinemas e supermercados.

O aplicativo foi pensando ao observar a carência de pesquisas e aplicativos que tratem a respeito da acessibilidade para pessoas no espectro autista, principalmente pelo carácter mais sensorial e atitudinal das suas necessidades.

## Funcionalidades

:heavy_check_mark: Login pelo Google ou Facebook.

:heavy_check_mark: Visualizar locais próximos.

:heavy_check_mark: Pesquisa de locais pelo nome e filtro por nota e tipo do local (cinema ou supermercado).

:heavy_check_mark: Visualização dos detalhes do local.

:heavy_check_mark: Visualização do conteudo relacionado ao autismo.

:heavy_check_mark: Avaliação da acessibilidade de um local.

:heavy_check_mark: Tradução para inglês automaticamente de acordo com a linguagem configurada no celular.

:heavy_check_mark: Armazenar dados de usuários, avaliações e locais usando o banco de dados da Cloud Firestore.

## Como usar

  ```
   Primeiramente, configure corretamente o ambiente de desenvolvimento Flutter em sua maquina, veja https://flutter.dev/docs/get-started/install
   
   - Clone this repository:
   $ git clone https://github.com/yanfalcao/GuiAutismo.git Guiautismo
   - Enter in directory:
   $ cd Guiautismo
   - For install dependencies:
   $ flutter pub get
   - Run the app: 
   $ flutter run
  ```
